# CEMS — Conference and Event Management Software

Conference and Event Management Software for a federal training facility, built on Salesforce Service Cloud + Sales Cloud with Agentforce.

## Overview

CEMS manages the full lifecycle of a federal training campus — hotel accommodations, conference facilities, on-site instruction, badging/access control, enrollment, grading, and scheduling.

## Platform

- Salesforce Service Cloud + Sales Cloud
- Agentforce (AI automation)
- Experience Cloud (portal)
- Salesforce Scheduler

## Status

Project kickoff — architecture and design in progress.
